  1. Signal fires in signal-monitor (a flip-level breach) → review the affected theme page.
  2. MDAEM pass (10 gates → 10 dims → tier → SA-Quant overlay → scenario EV) on any new name.
  3. File the row here + a falsifiable P-call in prediction-ledger (success/failure thresholds).
  4. On invalidation → mark EXIT, move to track-record with realized P&L.
  5. On resolve-by → score the P-call (✅/❌/◐), update track-record hit-rate.

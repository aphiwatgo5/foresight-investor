MDAEM v4.0 — Quantitative Extension

Definition: v4.0 extends the MDAEM v3.4 single-name decision system with a
quantitative signal layer that automates scoring of the dimensions where data is plentiful
(Valuation, Growth, Technical, Sentiment, Macro fit), leaving the qualitative dimensions
(Quality/Moat, Financial Health, Earnings Quality, Catalysts, Asymmetry) for operator judgment.
The gates, weights, tier lookup, SA-overlay, and 21 refinements are unchanged — v4.0 adds,
never replaces.

Scope & guardrail. v4.0 inherits v3.4's guardrail verbatim: decisions for the operator's
own book, never personalized advice to others. The quant layer is a research tool that surfaces
candidates and automates data work — it does not remove the operator's judgment from the loop.

Why v4.0 exists

v3.4's documented weaknesses (per mdaem-backtest-2026-06-28): high precision (loss-avoider),
lower recall (opportunity-capturer); manual Dim 6/7 scoring is inconsistent and time-consuming;
no systematic universe screening (the operator passed on names one-by-one). v4.0 closes all three:

  1. Automated daily screening — scripts/signals.py ranks all ~280 instruments daily; the
    top decile per style is surfaced for review (vs. the manual one-name-at-a-time v3.4 workflow).
  2. Quantitative Dim 3/5/6/7/9 scoring — scripts/mdaem_v4.py produces data-driven dimension
    scores, removing manual judgment from where data is unambiguous.
  3. Backtested strategy sleeves — the 4 styles each carry a measured hit-rate + alpha, so the
    operator knows which style works in which regime (vs. intuition-only).

What changed from v3.4 (and what didn't)

Unchanged (the v3.4 spine — do not break)

  • The 10 hard gates (asymmetry, catalyst, liquidity, accounting, bear case, SA-Quant,
    revisions, M&A ceiling, multiple, financing) — verbatim.
  • The 10 weighted dimensions and their weights (Quality 15% / Fin Health 10% / Valuation 15% /
    Earn Quality 10% / Growth 10% / Technical 10% / Sentiment 10% / Catalysts 10% / Macro 5% /
    Asymmetry 5%).
  • The tier lookup (85+ T1 / 70–84 MED-HIGH / 55–69 STARTER / 40–54 PASS / <40 REJECT).
  • The SA-Quant overlay ("when MDAEM and SA conflict, take the LOWER signal").
  • The 21 refinements (Refs A–O + 16–21) — all preserved as documented lessons.
  • The scenario math (Bull/Base/Bear → EV → asymmetry ratio ≥1.5:1).
  • The sizing modifiers (52w-low +fundamentals +1%, +500% 12mo −1–2%, sector >25% trim, etc.).

Added (the v4.0 quant layer)

  • 5 quantitative dimensions now auto-scored from the signal panel:
  • Dim 3 Valuation ← fwd P/E (inverted, capped at 100x as a Gate-9 proxy) + % off 52w high.
  • Dim 5 Growth ← revenue growth YoY (mapped: >50% → 10, >30% → 9, … <−15% → 2).
  • Dim 6 Technical ← momentum composite percentile (1m/3m/6m ret + SMA cross + RSI + relative strength).
  • Dim 7 Sentiment ← inverse-momentum (crowding proxy; analyst-net-upgrade to be wired in).
  • Dim 9 Macro fit ← the macro-rotation score (regime-conditioned: bull rewards beta, bear rewards low-vol).
  • 5 qualitative dimensions remain operator-scored (Quality, Fin Health, Earn Quality, Catalysts,
    Asymmetry) — these need judgment, not data. The quant layer leaves them blank for operator input.
  • 4 strategy sleeves — each is a backtested, rules-based composite (see strategy-sleeves):
    momentum / value / quality / macro-rotation. Each has measured CAGR, Sharpe, max drawdown, alpha.

The v4.0 scoring workflow

  1. Run scripts/signals.py → daily signal panel + sleeve percentile scores for all ~280 names.
  2. Run scripts/mdaem_v4.py → quant Dim 3/5/6/7/9 scores + composite_quant_only per name.
  3. Operator completes the 5 qualitative dims (Quality, Fin Health, Earn Quality, Catalysts, Asymmetry).
  4. Apply the 10 hard gates; any fail → REJECT (with documented reason mapped to a Ref).
  5. Composite total → tier → SA-overlay → sizing verdict.
  6. Tag the name with the best-fit strategy sleeve (a name can be in 2 sleeves).
  7. File in decision-register + prediction-ledger + track-record.

The 4 strategy sleeves (v4.0's second innovation)

See strategy-sleeves for the master table; backtest-results-2026-07-18 for the full metrics.
Backtested over 2021-07-17 → 2026-07-17 (5y, monthly rebalance, top-20 names per sleeve, equal-weight):

Momentum +29.4% 1.04 -40.5% +20.7 pts Bull (+100%/yr annualized in bull days)
Macro-rotation +22.7% 0.91 -37.4% +14.0 pts Adapts (rules shift by regime)

Key regime-split finding: Momentum is deeply regime-dependent — +100%/yr in bull days,
-46%/yr in bear days. The macro-rotation sleeve exists to dial momentum exposure up/down by
regime. Quality is the all-weather compounder (Sharpe ~1 with VOO-like drawdown).

Target portfolio allocation (from backtest-tuned weights)

Indicative blend (operator-adjustable based on conviction + regime):
- Quality / long-term compounder: 35–45% — the spine; lowest drawdown, best risk-adjusted.
- Momentum / trend: 20–30% — high-alpha but capped; reduce in bear regime.
- Macro-rotation: 15–25% — the regime-switching sleeve; flexes offense/defense.
- Value / contrarian: 10–15% — small dedicated sleeve; outperforms in recovery regimes.
- Cash (SGOV): 3–10% — dry powder for pullback adds (preserved from v3.4).

These weights plug into portfolio-blueprint's barbell on top of the existing meta-force
allocations.

Calibration: does the quant layer improve on manual Dim 6/7 scoring?

The honest answer (from the validation DB): the existing manual
Dim 6/7 scoring has ~75% factor accuracy with documented biases (reflexive small-cap dismissal,
outdated-narrative anchoring). The quant layer eliminates those biases by construction — it
scores every name the same way from the same data. Whether the quant scores are better
predictions is a hypothesis the v4.0 walk-forward tests; the backtest above shows the 4 sleeves
each outperform VOO over 5y, which is necessary-but-not-sufficient evidence.

The pre-registered test (feeding the operator's 2026-08-24 review): for each name where v3.4 manual
Dim 6/7 disagreed with the v4.0 quant score, did the quant or manual score predict the next-3-month
return better? Track in track-record going forward.

Data sources + known gaps

VIX/S&P/gold/BTC/DXY), all pandas-computed technicals (SMA/EMA/RSI Wilder/MACD/Bollinger/ATR/
Sharpe/drawdown). n/f flagged: quarterly fundamentals pre-2025 (only ~5 quarters deep —
use annual income_stmt for 5y); HY OAS pre-2023 (only 3y on FRED); FINRA margin debt (not on
FRED); SA-Quant grades (paywalled — stays operator overlay).
